# test-tool-1

A CLI tool built with TypeScript, Commander, and Consola.

## Installation

```bash
npm install
```

## Development

```bash
# Run in development mode
npm run dev

# Build the project
npm run build

# Run the built version
npm start
```

## Usage

```bash
# Display help
npm run dev -- --help

# Say hello
npm run dev -- hello
npm run dev -- hello "Your Name"

# Display info
npm run dev -- info
```
