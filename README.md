# CLI Test Tool

CLIツールを開発するためのテスト用ディレクトリ。

## 含まれるプロジェクト

- [wc](./wc) - 文字/行/単語数をカウントするCLIツール
- [simple-todo-cli](./simple-todo-cli) - シンプルなTODO管理CLI
- [simple-note-cli](./simple-note-cli) - シンプルなメモCLI
- [git-taiwa](./git-taiwa) - Gitコミットメッセージ支援ツール
- [whoami](./whoami) - ユーザー情報を表示するCLI
- [fake-weather](./fake-weather) - 天気情報CLI（開発中）
- [shell-list](./shell-list) - シェル一覧CLI
- [text-util](./text-util) - テキスト処理CLI
- [time](./time) - 日時処理CLI

## 開発

各プロジェクトは独立したnpmパッケージです。各ディレクトリで開発してください。

良いものができたら、レポジトリを分けて公開することを検討してください。
