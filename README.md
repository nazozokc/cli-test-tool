# test-tool2

A CLI tool built with TypeScript and Commander.js

## Installation

```bash
npm install
```

## Build

```bash
npm run build
```

## Development

Run the CLI in development mode:

```bash
npm run dev
```

## Usage

After building:

```bash
npm start -- [options]
```

Or if installed globally/linked:

```bash
test-tool2 [options]
```

## Available Commands

- `test-tool2 --version` - Display version information
- `test-tool2 --help` - Display help information

## Development Scripts

- `npm run build` - Compile TypeScript to JavaScript
- `npm run dev` - Run the CLI in development mode with ts-node
- `npm run watch` - Watch for changes and recompile automatically
- `npm start` - Run the compiled CLI
